# shared-types

> 🛡️ **纯类型定义包 (Pure Types Package)**
>
> 本项目的所有共享数据类型、接口定义及常量枚举均集中于此。旨在确保主进程 (Main)、预加载脚本 (Preload) 和渲染进程 (Renderer) 之间的类型安全与数据契约一致性。

## ⚠️ 核心原则

**本包仅用于定义类型，严禁包含任何业务逻辑。**

- ✅ **允许**: 接口 (`interface`)、类型别名 (`type`)、常量对象 (`const ... as const`)、枚举值定义。
- ❌ **禁止**: 函数实现、类实例化、异步操作、副作用代码、第三方库依赖（除类型依赖外）。
- 🚫 **零运行时**: 编译后不应产生任何可执行的 JavaScript 逻辑代码。
