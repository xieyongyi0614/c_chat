# 媒体预览需求分析与任务编排

## 背景

桌面端通过独立 Electron 预览窗口查看图片和视频；Web 端无法复刻 Electron 独立窗口语义。Web 端一期采用应用内 Lightbox 或路由弹层，复刻图片缩放、拖拽、旋转、切换和视频播放控制等核心体验。

本模块只规划 Web 端预览能力，不再新增独立 `apps/media_preview`。桌面端独立窗口需求详见 [MEDIA_PREVIEW_REQUIREMENTS.md](../requirements/MEDIA_PREVIEW_REQUIREMENTS.md)，Web 端取舍见 [09 桌面能力取舍](09_DESKTOP_CAPABILITY_MAP.md)。

## 当前基础

### 已具备

- 消息模块已有图片、视频消息展示入口。
- 上传模块会为发送中媒体提供 File/Blob 和 object URL。
- 历史消息可使用服务端 `fileUrl`。
- `packages/chat_ui` 提供 Dialog、Button、Tooltip、ScrollArea、Spinner 等基础 UI。
- 桌面端已有 `MediaPreviewItem` / `MediaPreviewPayload` 类型思路，可作为 Web 预览数据结构参考。

### 主要缺口

- Web 端需要应用内 Lightbox 状态管理。
- Web 端需要图片查看器和视频播放器组件。
- Web 端不使用 `filePath` 和 `ReadLocalFile`，需要统一远端 URL 与 File/Blob object URL。
- object URL 生命周期需要明确释放。
- 快捷键、焦点管理和 a11y 需要由弹层组件承载。

## 目标

### 一期目标

- 点击图片消息打开应用内 Lightbox。
- 点击视频消息打开应用内 Lightbox 播放。
- 支持图片缩放、拖拽、旋转、切换、还原。
- 支持视频播放、暂停、进度、音量、静音、倍速。
- 支持常用快捷键。
- 支持远端资源和发送中本地 File/Blob 资源。

### 二期方向

- 会话级媒体浏览器。
- 普通文件预览。
- 图片复制、保存、查看原图。
- 视频截图、画中画、全屏。
- 预加载前后媒体。

## 产品需求

### 图片预览

用户点击图片缩略图后，在应用内 Lightbox 查看大图。

验收标准：

- 单张图片点击后打开 Lightbox。
- 图片组点击任意一张后从当前索引进入。
- 支持上一张、下一张。
- 支持滚轮缩放。
- 支持拖拽移动。
- 支持旋转和双击还原。
- 支持加载中、失败、重试状态。

### 视频预览

用户点击视频消息后，在 Lightbox 中播放。

验收标准：

- 不跳转外部链接。
- 支持播放、暂停。
- 支持进度拖动。
- 支持音量、静音、倍速。
- 展示文件名、时长、当前时间。
- 关闭预览时停止播放。
- 加载失败或格式不支持时展示错误。

### 快捷键

预览弹层支持常用键盘操作。

验收标准：

- `Esc` 关闭。
- 左右方向键切换图片或视频快退/快进。
- `+` / `-` 缩放图片。
- `0` 还原图片。
- `Space` 播放或暂停视频。
- 弹层打开后焦点留在预览区域，关闭后焦点回到触发元素。

### 资源解析

预览同时支持远端历史资源和发送中本地资源。

验收标准：

- 有 `fileUrl` 时使用远端 URL。
- 发送中 File/Blob 使用 object URL。
- Web 端不使用 `filePath`。
- Web 端不调用 `ReadLocalFile`。
- 切换媒体或关闭弹层时释放 object URL。

### 容器体验

Lightbox 使用深色沉浸式背景，不影响聊天主布局。

验收标准：

- 弹层覆盖应用视口。
- 媒体内容居中。
- 操作栏弱化但可发现。
- 图片缩放限制在合理范围。
- loading、empty、error 状态完整。
- 弹层内部滚动和聊天页滚动不冲突。

## Web 适配要点

- 独立 Electron 预览窗口替换为应用内 Dialog / 路由弹层 / 自定义 Lightbox 容器。
- 基础弹层能力优先用 `chat_ui` / shadcn Dialog，不手写 focus trap 和外部点击关闭。
- `ImageGroup` 和 `VideoMessage` 只负责组装预览数据并打开预览。
- Web 端 `sourceWindowId` 无意义，不进入一期逻辑。
- 远端视频体验依赖服务端是否支持 Range 请求。

## 风险点

- object URL 不释放会造成内存泄漏。
- 远端视频如果不支持 Range，大文件拖动和加载体验会变差。
- 大图缩放可能造成卡顿，必要时限制最大尺寸或延迟加载。
- 快捷键需要避免和浏览器默认快捷键、输入框焦点冲突。
- 不要为了复刻桌面端独立窗口而引入 `window.open` 主流程。
